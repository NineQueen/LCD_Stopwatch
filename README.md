# LCD_Stopwatch
TM1119 LCD_Stopwatch

SW1 enter setting mode 

SW2 run the timer

Entering setting mode

SW1 change the mode (second to min, min to hour, hour to start mode)

SW2 add the timer

Run the timer

SW1 stop the timer

SW2 reset the timer (back to the start mode)

Time over

SW2 reset the timer
